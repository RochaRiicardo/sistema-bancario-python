# sistema-bancario-python

Projeto simples em Python para praticar logica de programacao, funcoes, estruturas condicionais e controle de fluxo.

## Arquivos

- `sistema bancario v1.2.py`: sistema bancario com operacoes basicas.
- `Biblioteca v1.2.py`: sistema de biblioteca feito em Python.

## Como executar

1. Tenha o Python instalado no computador.
2. Baixe este repositorio.
3. Execute o arquivo desejado pelo terminal:

```bash
python "sistema bancario v1.2.py"
```

ou

```bash
python "Biblioteca v1.2.py"
```
